## 荟诚边缘计算SDK支持人脸识别和车牌识别
# 人脸识别：
- 人脸查找
- 人脸关键点提取
- 人脸对齐
- 人脸特征提取
- 人脸库特征搜索
- 单双目活体检测
# 车牌识别
- 车牌定位
- 车牌识别
